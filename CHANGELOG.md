# Changelog

All notable changes to Radix Free are documented here.

---

## [1.0.0] | 2026-06-30

Initial public release of **Radix Free**.

### What's included
- 39 snap positions accessible via grouped dropdown menus | Faces (6), Vertices (8), Edge Midpoints (12), Centers (4), Global Centers (Geometry / BBox / Mass), Cursor, World Zero
- BBox Handle mode | 26 clickable handles in the viewport with hover-enlargement (1.55×)
- Origin → World Zero | one button, no modal
- Snap Reference controls | snap source (Mesh / BBox / Cursor), orientation (Local / Global), and front face axis
- Alt+Q pie menu | 8 quick snap positions from anywhere in the viewport
- Blender 4.5+ / 5.x compatible (extension format with `blender_manifest.toml`)
